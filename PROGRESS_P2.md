### 任务 2.1 详细进度：Docker工具集

**状态**: ⬜ 未开始  
**计划开始日期**: 第3周  
**计划完成日期**: 第3周

#### 任务描述

提供Docker容器管理能力，支持容器操作、镜像管理和compose编排。

#### 技术方案

```
tools/docker/
├── __init__.py
├── docker_ps.py      # 列出运行中的容器
├── docker_logs.py    # 查看容器日志
├── docker_exec.py    # 在容器中执行命令
├── docker_build.py   # 构建Docker镜像
└── docker_compose.py # Docker Compose操作
```

#### 安全考虑

- 危险命令拦截（rm、kill等）需在白名单模式下运行
- 支持只读模式，禁止修改操作

#### 验收标准

- [ ] docker_ps 工具可用
- [ ] docker_logs 工具可用
- [ ] docker_exec 工具可用
- [ ] docker_build 工具可用
- [ ] docker_compose 工具可用
- [ ] 安全机制（白名单、只读模式）实现
